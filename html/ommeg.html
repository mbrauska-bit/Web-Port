Mark Brauska-om meg

Jeg er en motivert elev på 17 år som går på VG1 Infromasjonteknologi og medieproduksjon ved Bleiker videregående skole. Helt siden startet jeg på denne linjen har jeg vært interessert i å lære om webutvikling og grafisk design.Gjennom skoleoppgaver har jeg jobbet mye med koding og grafisk design. Jeg har en del forståelse for hvordan HTML og CSS brukes til struktur og design, og jeg bruker Javascript for å gjøre nettsiden mer funksjonelle og interakriv. Ved siden av skolen har jeg arbeidserfaring som dommer på ishockey. Denne rollen har jeg lært meg å vurdere spillerne, å kommunisere med andre dommere og å ta beslutninger under press.

Ferdigheter

  • Utdanning: VG1 Informasjonteknologi og medierproduksjon, Bleiker videregående skole.
  • Webutvikling: Jeg lærer å bygge strukturerte og responsive nettsider ved hjelp av HTML, CSS og Javascript.Mens HTML og CSS bestemmer hvordan siden ser ut, bruker jeg JavaScript til å gjøre nettsidene levende. Jeg lærer å bruke det til å programmere interaktive elementer – som for eksempel å lage bildekaruseller, mørk/lys-modus-knapper, nedtrekkshovedmenyer (burgermenyer) eller animasjoner som reagerer når brukeren klikker på noe.
  • Grafisk design: Jeg lærer å bruke Adobe Photoshop og Adobe Illustrator for å lage grafiske produkter og redigere bilder. Jeg lærer å lage logoer, plakater, og annet visuelt materiell ved hjelp av disse verktøyene.
  • Linux: Vi har akkurat begynt med Linux på skolen, og jeg synes det er spennende å lære hvordan man bruker terminalen og styrer systemet med kommandoer.
